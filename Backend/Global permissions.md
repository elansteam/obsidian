Глобальные  права доступа, которые агрегируют в себе роли. Отображение этих прав находится в `auth.utils.Permissions` в файлах проекта
# CREATE_USER 
  
Создание пользователя
# CREATE_GROUP
Создание группы
# CREATE_ROLE
Создание роли
# CHANGE_USER_ROLES
Изменение ролей пользователя
# DELETE_USER
Удаление пользователя
